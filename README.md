# ManutSGQ

Sistema de controle de manutenções preventivas e corretivas com rastreabilidade para apoio ao Sistema de Gestão da Qualidade (SGQ).

## Objetivo
Desenvolver um sistema para gerenciar manutenções de equipamentos e máquinas, permitindo controle de planos preventivos, registros corretivos, histórico de intervenções, indicadores e rastreabilidade de informações para auditoria e melhoria contínua.

## Escopo inicial
- Cadastro de equipamentos
- Cadastro de planos de manutenção preventiva
- Abertura de manutenções corretivas
- Registro de execução de manutenção
- Histórico por equipamento
- Indicadores de manutenção
- Controle de status e responsáveis
- Apoio à rastreabilidade conforme SGQ

## Perfis de usuário
- Operador
- Técnico de manutenção
- Gestor/Líder
- Qualidade/SGQ
- Administrador
